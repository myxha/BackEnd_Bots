# Telegram Food Bot (BMR & TDEE Calculator)

Бот для розрахунку денної норми калорій (BMR та TDEE) на основі профілю користувача. Побудований на **Bun** + **grammY**.

## Функції
- Покрокове створення профілю (/set_profile)
- Валідація введених даних (вік, зріст, вага)
- Розрахунок BMR (формула Міффліна-Сан Жеора)
- Розрахунок TDEE (з урахуванням рівня активності)
- Перегляд профілю (/my_profile)

## Структура проекту
- `index.ts` — основний файл з логікою бота
- `package.json` — залежності та скрипти
- `.env` — токен бота (створюється вручну)

## Як запустити

1. **Встановити залежності:**
   ```bash
   bun install
   ```

2. **Налаштувати токен:**
   - Створіть файл `.env` у корені проекту.
   - Скопіюйте вміст з `.env.example` та вставте свій токен від [@BotFather](https://t.me/BotFather).
   ```text
   BOT_TOKEN=your_token_here
   ```

3. **Запустити бота:**
   ```bash
   # Режим розробки (з автоперезавантаженням)
   bun run dev

   # Звичайний запуск
   bun run start
   ```

## Використані технології
- [Bun](https://bun.sh/) — швидкий рантайм для JavaScript/TypeScript
- [grammY](https://grammy.dev/) — фреймворк для Telegram ботів
- [@grammyjs/conversations](https://grammy.dev/plugins/conversations) — плагін для створення діалогів
