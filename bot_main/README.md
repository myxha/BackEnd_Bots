# Telegram Bot (Bun + grammY)

Цей проект — це простий Telegram-бот, написаний на TypeScript з використанням рантайму Bun та бібліотеки grammY.

## 🚀 Швидкий запуск

1.  **Встановіть залежності:**
    ```bash
    bun install
    ```

2.  **Налаштуйте токен:**
    - Скопіюйте `.env.example` у `.env`
    - Отримайте токен у [@BotFather](https://t.me/BotFather) та вставте його у файл `.env`:
    ```text
    BOT_TOKEN=ваш_токен_тут
    ```

3.  **Запустіть бота в режимі розробки:**
    ```bash
    bun run dev
    ```

## 🛠 Команди бота

- `/start` — Привітальне повідомлення.
- `/help` — Список доступних команд.
- `/joke` — Випадковий жарт про програмістів.

## 📝 Особливості
- Реалізовано на TypeScript.
- Використовує `dotenv` для безпечного зберігання токенів.
- Обробляє текстові привітання ("привіт", "hello").
- Підказує команду `/help`, якщо написати "help" текстом.
- Має механізм випадкових відповідей.
